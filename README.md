## Features
- **AI Templates** - Intelligent document generation based on jurisdiction and case type.
- **Smart Collaboration** - Real-time editing and commenting for legal teams.
- **Strict Security** - Enterprise-grade encryption for sensitive legal data.
- **Version Control** - Track changes and maintain document history effortlessly.

## Installation
```bash
npm install
```

## Quick Start
```bash
npm run dev
```

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS
- **State Management**: React Hooks & Context API
- **Design**: Shadcn UI / Lucide Icons

> [!TIP]
> Ensure you have your environment variables set up in `.env.local` for AI services to function correctly.
